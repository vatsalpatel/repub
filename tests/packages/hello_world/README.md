# Hello World

A simple Dart package that provides a hello world function.

## Features

* Simple hello world functionality
* Easy to use API
* Well documented

## Getting started

Add this to your package's pubspec.yaml file:

```yaml
dependencies:
  hello_world: ^1.0.0
```

## Usage

```dart
import 'package:hello_world/hello_world.dart';

void main() {
  print(helloWorld());
}
```
